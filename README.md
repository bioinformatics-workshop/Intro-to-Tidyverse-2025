This workshop will provide a brief introduction to data wrangling using the Tidyverse R package. This workshop is intended for beginner users with some familiarity with R. Topics covered in this workshop includes:

- Basic introduction to the Tidyverse
- Importing data
- Cleaning data
- Process data
